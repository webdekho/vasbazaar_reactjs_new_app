# Customer App Claude Guide

This is the React + CRA + Capacitor customer app. Start with `AGENTS.md` in this folder for the compact map.

Fast paths:

- Routes: `src/customer/CustomerModernRoutes.jsx`
- Screens: `src/customer/pages/`
- Components: `src/customer/components/`
- Services: `src/customer/services/`
- Context: `src/customer/context/`
- Styling/theme: `src/customer/customerModern.css`, `src/customer/theme.js`

Use `npm run build` to verify compilation.
